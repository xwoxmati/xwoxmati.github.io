Detección de anomalías en consumo energético

Rol: Diseño y modelado · 2025

Descripción: Pipeline de análisis para identificar consumos atípicos en series temporales (simuladas o abiertas) con el fin de alertar ineficiencias o fallas.

  • Objetivos: Construir un sistema de alerta temprana para detectar desviaciones y apoyar decisiones de mantenimiento.
  • Tecnologías: Python (pandas, scikit‑learn), Isolation Forest / One‑Class SVM, visualización con Matplotlib.
  • Resultados: Mejora en la detección de outliers y definición de umbrales operativos interpretables.
  • Reflexión: La estandarización del preprocesamiento (resampleos, z‑score) y métricas de evaluación fue clave para resultados consistentes.

Estimación de producción FV con modelos de regresión

Rol: Modelado predictivo · 2025

Descripción: Modelo de regresión para estimar producción diaria de un sistema FV a partir de variables meteorológicas (irradiancia, temperatura, nubosidad) y parámetros del sistema.

  • Objetivos: Comparar modelos base (Linear, Random Forest, XGBoost) y evaluar error (MAE/RMSE) para escenarios climáticos.
  • Tecnologías: Python (scikit‑learn, xgboost), validación cruzada, ingeniería de características (lags/rolling).
  • Resultados: Reducción de error respecto a baseline climatológico; aprendizaje sobre sensibilidad a la calidad de datos.
  • Reflexión: El feature engineering temporal tuvo mayor impacto que la complejidad del modelo en sí.

E‑learning en Moodle con recomendación básica

Rol: Implementación y prototipo ML · 2025 (en curso)

Descripción: Implementación de un curso en Moodle (estructura de unidades, cuestionarios y tracking). Prototipo simple de recomendación de contenidos (popularidad/recencia) para sugerir lecturas y quizzes.

  • Objetivos: Crear curso navegable con evaluación y un módulo de sugerencias iniciales.
  • Tecnologías: Moodle, Python (pandas), exportación/importación de datos de actividad para generar recomendaciones.
  • Resultados (esperados): Incremento de engagement en ítems sugeridos; base para un recomendador colaborativo.
  • Reflexión: Alinear objetivos pedagógicos con métricas de uso es clave para recomendaciones útiles.
